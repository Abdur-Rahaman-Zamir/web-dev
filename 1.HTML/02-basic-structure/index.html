This is a simple HTML document to demonstrate the basic structure of an HTML page.

Key Elements

  • DOCTYPE: Defines the document type and version of HTML.
  • html: The root element of an HTML page.
  • head: Contains meta-information about the document.
  • title: Sets the title of the document, shown in the browser tab.
  • body: Contains the content of the document that is displayed in the browser.
